# i3-ipc

A Crystal interface for I3 IPC.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  i3-ipc:
    github: RyanScottLewis/i3-ipc
```

## Usage

```crystal
require "i3"
```

See `examples/` directory for more usage examples.
